# Hackathon Project--Topic 2 : AI-Generated Fitness Tracker
This project simulates the use of a fitness tracker that generates personalized workout plans and tracks progress over time. Using a dataset of fitness tracking data from Kaggle, it provides workout recommendations for users based on their prior workout history, and shows a variety of analyses and visualizations of the data to show insights on users' fitness conduct and results.

# Tech Stack:
Python

# How to Run the Project
The project runs completely within the ProjectNotebook.ipynb Jupyter Notebook file. The file demonstrates use of the workout recommendation function. One can use that function within the file to produce additional workout recommendations by entering new lines of code calling the function with additional parameters.

Dependencies:
pandas, numpy, scipy, matplotlib, seaborn.

The project is contained in the ProjectNotebook.ipynb file and uses the evan_fitness.csv file as a data source.
To start the project, download ProjectNotebook.ipynb and evan_fitness.csv into the same folder, and run ProjectNotebook.ipynb in any program that supports Jupyter Notebooks.

# Main Features:
- Function to recommend workouts based on users' workout history.
- Analyses and visualizations of fitness tracking data, along with conclusions and insights that follow.
